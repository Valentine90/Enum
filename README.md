## About
![lANG](https://img.shields.io/badge/LANG-RUBY-red?style=for-the-badge&logo=appveyo)

[Enumeration type](https://docs.microsoft.com/pt-br/dotnet/csharp/language-reference/builtin-types/enum) for Ruby.

## Examples
```Ruby
Sex = enum %w(
  MALE
  FEMALE
)
```
Or:
```Ruby
Dir = enum %w(
  DOWN_LEFT  1
  DOWN_RIGHT 3
)
```
